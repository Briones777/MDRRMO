First Aid SOP / Unang Lunas SOP

DRSABC / DRSABC

  • Danger / Kaligtasan
  • Response / Pagtugon (Check consciousness / Suriin kamalayan)
  • Send Help / Tawag ng Tulong (911)
  • Airway / Daan ng Hangin
  • Breathing / Paghinga (10s)
  • Circulation / Sirkulasyon (CPR kung kinakailangan)

Bleeding / Pagdurugo

  • Apply direct pressure / Direktang pagpindot
  • Elevate limb / Iangat ang bahagi ng katawan
  • Use tourniquet if severe / Gumamit ng tourniquet kung malubha

Burns / Paso

  • Cool burn under running water / Palamigin sa tubig
  • Cover with sterile cloth / Takpan ng malinis na tela
  • Do not apply ointments / Huwag maglagay ng pomada

Fracture / Bali

  • Immobilize limb / I-immobilize ang bahagi ng katawan
  • Apply splint if available / Gumamit ng splint kung mayroong
  • Elevate and ice / Iangat at palamigin

Choking / Pagkasakal

  • Encourage coughing / Hikayatin na umubo
  • Heimlich maneuver / Heimlich maneuver
  • Call 911 if obstruction persists / Tawag 911 kung hindi mawala

Faint / Panlabo

  • Lay patient flat / Ihiga ang pasyente
  • Elevate legs / Iangat ang mga paa
  • Check breathing / Suriin ang paghinga

Dizziness / Pagkahilo

  • Sit or lie down / Umupo o humiga
  • Drink water / Uminom ng tubig
  • Monitor symptoms / Obserbahan ang kalagayan

CPR / CPR

  • Chest compressions 100-120/min / Pindot sa dibdib 100-120/min
  • Depth 5-6 cm / Lalim 5-6 cm
  • Ratio 30:2 / Ratio 30:2
  • Continue until help arrives / Ipagpatuloy hanggang dumating ang tulong

Shock / Pagkamaingay

  • Lay patient flat / Ihiga ang pasyente
  • Keep warm / Panatilihing mainit
  • Monitor breathing / Obserbahan ang paghinga

Seizure / Pangingisay

  • Protect head / Protektahan ang ulo
  • Clear surrounding objects / Alisin ang mga bagay sa paligid
  • Do not restrain / Huwag pigilan ang kilos
  • Call 911 if prolonged / Tawag 911 kung matagal

Poisoning / Pagkalason

  • Check what was ingested / Alamin kung ano ang nainom
  • Call poison center / Tawag sa poison center
  • Do not induce vomiting unless instructed / Huwag pilitin magsuka
